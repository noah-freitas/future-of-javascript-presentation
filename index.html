Data Structures

Array Functions

Moving Beyond for

  • forEach
  • map
  • reduce
  • filter
  • Others

forEach

Executes a function for each item in an array.

The function is passed three parameters: the current value, the index, and the complete array.

							
var arr = [1, 2, 3];

var print = function (item) {
  console.log(item);
};

arr.forEach(print);

// Logs out:
//   1
//   2
//   3

map

Creates a new array by calling a function on each element of an array.

The function is passed three parameters: the current value, the index, and the complete array.

							
var arr = [1, 2, 3];

var times2 = function (item) {
  return item * 2;
};

var times2Array = arr.map(times2);

// times2Array now is:
//   [2, 4, 6]

reduce

Reduces an array to a single value by calling a function for each element in the array.

The function is passed four parameters: the previous value, the current value, the index, and the complete array.

							
var arr = [1, 2, 3];

var sum = function (item1, item2) {
  return item1 + item2;
};

var total = arr.reduce(sum);

// total is 6.

filter

Creates a new array with only those elements for which your function returns true.

The function is passed three parameters: the current value, the index, and the complete array.

							
var arr = [1, 2, 3, 4, 5, 6];

var isEven = function (item) {
  return item % 2 === 0;
};

var evens = arr.filter(isEven);

// evens is:
//   [2, 4, 6]

Others

  • indexOf
  • every
  • some

indexOf

Searches an array for a value. Returns the first index at which it appears or -1 if the value is not in the array.

Also: lastIndexOf

every

Returns true if your function returns true for each element in the array.

some

Returns true if your function returns true for any element in the array.

Generator Functions

Functions that return an infinite sequence, with lazy computation. 

                            
function* naturalNumbers() {
  num = 0;
  while (true) {
    num += 1;
    yield num;
  }
}
naturalNumbers(); // 1
naturalNumbers(); // 2

Generator Expressions

More efficient function arguments.

                            
// Harmony example
(xhrGet(url) for (url of getURLs()))

Iterators

Better for in loops.

                            
var getClassRoster = function() {
  return {
    iterator: function () {
      return {
        elements: roster.students,
        index: 0,
        next: function () {
          if (this.index >= this.elements.length) {
            throw StopIteration;
          }
          this.index += 1;
          return this.elements[this.index];
        }
      };
    },
    // ...
  };
};

for (var student of getClassRoster()) {

Array Comprehensions

Succintly create one array from another.

map example

                            
var arr = [1, 2, 3];

var times2Array = [x * 2 for (x of arr)];

filter example

                            
var arr = [1, 2, 3, 4, 5, 6];

var evens = [x for (x of arr) if (x % 2 === 0)];

forEach example

                            
var arr = [1, 2, 3];

[console.log(x) for (x of arr)];

Maps & Sets

Easy key/value or key collections of any value type.

Set example

                            
var playerGuesses = new Set();
playerGuesses.add(1);
playerGuesses.add("two");

playerGuesses.has(1); // true
playerGuesses.has(2); // false

Map example

                            
var players = [{ name: "Mary" }, { name: "Bob" }];
var playerGuesses = new Map();
playerGuesses.add('heads', players[0]);
playerGuesses.add('tails', players[1]);

var flip = Math.random() > .5 ? 'heads' : 'tails';
var winner = playerGuesses.get(flip);
var winningName = winner.name;
// winningName is "Mary" or "Bob"

Scope & Value

let

Provides a way to create sub-scopes within a function.

The let keyword can be used in three forms:

  • block
  • statement
  • expression

Block Example

							
var a = 5;

for (let ii = 0; ii < 5; ii += 1) {
  let a = 3;
  console.log(ii + ': ' + a);
}

// Logs out:
//   0: 3
//   1: 3
//   2: 3
//   3: 3
//   4: 3

Expression & Statement Examples

							
var a = 5;
let (a = 3) console.log(a);  // Logs 3
console.log(a);              // Logs 5

var print = function (item) {
  console.log(item);
};

print(let (a = 4) a);        // Logs 4

Parameter Changes

  • Default Parameters
  • Rest Parameters
  • Spread Operator

Default Parameters

                            
var print = function (item = "nothing") {
  console.log(item);
};
print(); // logs "nothing."

Rest Parameter

Simple variable argument functions.

Turning arguments into a real array.

                            
var push = function (arr, ...items) {
  for (var ii = 0; ii < items.length; ii += 1) {
    arr.push(items[ii]);
  }
};

var numbers = [1, 2, 3];
push(numbers, 4, 5, 6);
// numbers is now [1, 2, 3, 4, 5, 6]

Spread Operator

Pass the items in an array to a function as individual parameters.

                            
var numbers = [1, 2, 3, 4, 5];
var largest = Math.max(...numbers);
// largest is 5

Modules

Simpler namespacing and code sharing.

Simple Module and Client

                			
// Module file.
module 'stringUtilities' {
  export function trim(str) {
    // trims whitespace from the string.
  }
}

// Client file.
import { trim } from 'stringUtilities';

var myString = "  abc d   ";
var trimmed = trim(myString);
// trimmed is "abc d"

Syntax Changes

Destructuring

Breaks apart an array or object during assignment (or other value bindings.)

Simple swap example

							
var a = 1;
var b = 3;

[a, b] = [b, a];

// b is 1 and a is 3.

Break apart a function return value

							
var firstName, age;

var getPerson = function () {
  return {
    name: {
      first: 'John',
      last: 'Doe'
    },
    age: 42
  };
};

{ name: { first: firstName }, age: age } = getPerson();

// firstName is 'John', age is 42

Break apart a function parameter

							
var person = {
  name: {
    first: 'John',
    last: 'Doe'
  },
  age: 42
};

var printFirstName = function ({ name: { first: firstName } }) {
  console.log(firstName);
};

printFirstName(person);  // Logs 'John'

Object.create

  • Simpler prototypal inheritance syntax.
  • New control over an object's properties.
    • Data Properties
    • Accessor Properties

Prototypal Inheritance

							
var videoPlayerPrototype = {
  play: function () {
    if (this.video) {
      console.log("Playing " +  this.video);
    }
  }
};

var player = Object.create(videoPlayerPrototype);
player.video = "Crockford on JavaScript";
player.play();
// Logs "Playing Crockford on JavaScript"

Controlling Data Properties

							
var player = Object.create(videoPlayerPrototype, {
  copyRight: {
    configurable: false,
    enumerable: false,
    value: 'SRJC',
    writable: false
  }
});
player.copyRight = 'Thief';
delete player.copyRight;

console.log(player.copyRight); // Logs 'SRJC'

Accessor Properties

							
var videoTitle;
var player = Object.create(videoPlayerPrototype, {
  video: {
    configurable: false,
    enumerable: true,
    get: function () {
      return videoTitle;
    },
    set: function (newTitle) {
      if (/crockford/ig.test(newTitle)) {
        title = newTitle;
      } else {
        alert('Rejected!');
      }
    }
  }
});

player.video = "Crockford on JavaScript"; // Works
player.video = "Avatar"; // alerts "Rejected!"

Immutable Objects

Prevent changes to an object, with varying levels of severity

  • Object.preventExtensions
  • Object.seal
  • Object.freeze

Object.preventExtensions

Can't add any new properties to the object.

See also: Object.isExtensible

Object.seal

Can't change the configuration of any properties.

See also: Object.isSealed

Object.freeze

Can't change anything about the object.

See also: Object.isFrozen

Object.observe

Add an event listener for any changes to an object.

							
var myObject = {};

var changeLog = function (changed) {
  console.log(changed);
};

Object.observe(myObject, changeLog);
							
						
							
myObject.id = "X123";

// Logs:
// { type: "new", object: myObject, name: "id" }

myObject.id = "1234";

// Logs:
// { type: "updated", object: myObject, name: "id", oldValue: "X123" }

Proxies

Create fully programmable interfaces for your Objects.

Many features natively supported by the language are "trappable".

Add an extra property to your object enumeration.

							
var propsPlusOne = Proxy({
  keys: function (target) {
    var keys = Object.keys(target);
    keys.push("secretKey");
    return keys;
  },
  get: function (target, name, receiver) {
    if (name === "secretKey") {
      return "You found it!";
    } else {
      return target[name];
    }
  }
}, {
  // rest of object.
});

DB interface.

							
var db = function () {
  return Proxy({
    get: function (target, name, receiver) {
      var db = JSON.parse(window.localStorage.db);
      return db[name];
    },
    set: function(target, name, val, receiver) {
      var db = JSON.parse(window.localStorage.db);
      db[name] = val;
      window.localStorage.db = JSON.serialize(db);
      return true;
    }
  }, {});
};
